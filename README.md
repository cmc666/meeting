# meeting

#### 介绍
该项目我对微服务架构学习的项目，通过分版本开发，最终完成一个能够应对实际生产环境的dubbo微服务项目

#### 软件架构
当前项目是一个基于dubbo的微服务前后端分离的项目架构计划使用的架构为：

1. 前端：vue,element-ui
2. 后端：
    
-     1. 基础架构：SpringBoot+Mybatis-Plus+Dubbo
-     2. 注册中心：zookeeper
-     3. 分布式事务框架：seata
-     4. 服务治理：senital
-     5. 接口文档：Swagger
-     6. 数据库：MySql
-     7. 消息中间件：RabbitMQ
-     8. 分布式缓存：Redis
-     9. 分库分表中间件：ShardingSphere-Jdbc
