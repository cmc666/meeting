# meeting

#### 介绍
该项目我对微服务架构学习的项目，通过分版本开发，最终完成一个能够应对实际生产环境的dubbo微服务项目

#### 软件架构
当前项目是一个基于dubbo的微服务前后端分离的项目架构计划使用的架构为：
前端：vue,element-ui
后端：
基础架构：SpringBoot+Mybatis-Plus+Dubbo
注册中心：zookeeper
分布式事务框架：seata
服务治理：senital
接口文档：Swagger
数据库：MySql
消息中间件：RabbitMQ
分布式缓存：Redis
分库分表中间件：ShardingSphere-Jdbc
